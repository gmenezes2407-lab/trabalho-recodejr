# 🔐 Interface de Login - Recode Jr

Projeto fullstack desenvolvido para o processo seletivo/trabalho da **Recode Jr**, consistindo em uma interface de login completa com front-end e back-end integrados.

## 📋 Funcionalidades

- Tela de login responsiva e moderna
- Validação de campos (usuario e senha)
- Conexão entre front-end e back-end
- Feedback visual para o usuário (erros, carregamento, sucesso)

## 🚀 Tecnologias Utilizadas

### Front-end
- HTML5
- CSS3
- JavaScript

### Back-end
- Node.js
- Express
